# Pizzeria

## About

A custom WordPress theme built from scratch based on a provided design, for a fictional pizzeria. The theme includes multiple page templates, custom typography, and conditional styling.

## Features

- Custom theme built from scratch (no parent theme)
- Custom page templates — Start page, Contact, Margherita, Salame, Vegetale
- header.php / footer.php with get_header() / get_footer()
- body_class() for conditional CSS per page-ID
- functions.php — admin bar hidden by filter
- Custom @font-face typography (FliegeMono Regular & Bold)
- CSS overlay with ::after and mix-blend-mode
- Dynamic page links via get_permalink()
- Asset files (SVG, JPG, WOFF2) organised in assets/ and fonts/

## Tech

WordPress, PHP, CSS, HTML
